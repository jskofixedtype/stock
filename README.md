# 무한매수법 대시보드 (TQQQ)

TQQQ 무한매수법의 **매수/매도 가격·수량을 자동 계산**하는 모바일 우선 웹 대시보드.
빌드 도구 없이 브라우저에서 바로 실행되며, 상태는 브라우저에 저장됩니다.

계산 규칙과 용어 정의는 [`CLAUDE.md`](CLAUDE.md), 설계는 [`PLAN.md`](PLAN.md) 참고.

## 주요 기능

- **설정 입력**: SEED, 분할수(40/30/20), 목표수익률, 평단, 수량
- **실시간 시세**: Finnhub API로 TQQQ 현재가/전일종가 자동 조회 (실패 시 수동 입력 폴백)
- **요약**: 일일매수금액·총매수금액·진행률·회차·매도점·별퍼센트·별퍼센트 매도점·남은 예산
- **MTS 주문 정보**: MTS에 그대로 입력할 매수/매도 주문 (주문유형·지정가·수량)
- **기준가 차트**: 평단·매도점·별퍼센트 가격·현재가 시각화
- **거래 이력**: 체결 기록 추가 시 평단·수량·진행률·실현손익 자동 재계산 (localStorage 저장)

## 실행 방법

정적 파일이므로 다음 중 아무 방법이나 사용합니다.

- **로컬**: `index.html`을 브라우저로 엽니다.
- **로컬 서버**(권장, API 호출 안정적): `python3 -m http.server` 후 `http://localhost:8000`
- **모바일(안드로이드 등)**: 아래 GitHub Pages 배포 후 폰 브라우저로 접속.

## Finnhub API 키 발급

실시간 시세 자동조회에는 무료 API 키가 필요합니다.

1. <https://finnhub.io> 가입 (무료)
2. 대시보드에서 발급된 **API Key** 복사
3. 앱의 **Finnhub API 키** 입력란에 붙여넣기 → localStorage에 저장되어 다음 접속 시 자동 적용

> 무료 티어는 분당 60회 제한이며 브라우저(CORS) 호출을 지원합니다.
> 키가 없거나 호출이 실패하면 **수동 현재가/전일종가** 입력값을 사용합니다.

## GitHub Pages 배포 (모바일용)

1. 저장소 **Settings → Pages**
2. Source: 원하는 브랜치(예: 기본 브랜치) / 루트(`/`) 선택 후 저장
3. 발급된 URL을 안드로이드 브라우저로 접속 (홈 화면에 추가하면 앱처럼 사용 가능)

## 파일 구조

| 파일 | 역할 |
|---|---|
| `index.html` | 대시보드 마크업 (설정/시세/요약/MTS주문/차트/이력) |
| `styles.css` | 모바일 우선 반응형 + 다크/라이트, 차트 색상 |
| `calc.js` | 계산 엔진(순수 함수) — 파생값·오늘의 주문 |
| `history.js` | 거래 이력 + 평단/수량/진행률/실현손익 재계산 |
| `chart.js` | 기준가 차트 (인라인 SVG) |
| `price.js` | Finnhub 시세 조회 |
| `storage.js` | localStorage 저장/복원 |
| `app.js` | 이벤트/상태/렌더 통합 (진입점) |
