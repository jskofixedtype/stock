## 무한매수법 계산기

## 무한매수법 용어정리
 1. SEED: 무한매수법 총 매수 예산
 2. 분할수: 일일 매수금액을 설정하기 위해 전체매수 금액을 나누는 값 (40, 30, 20)
 3. 일일매수금액 : SEED / 분할수 
 4. 평단 : 현재 매수되어 보유중인 주식의 평균단가
 5. 수량: 현재 보유 중인 주식의 총 갯수
 6. LOC  : Limit on Close - 장마감 지정가
 7. 목표수익률(%) : 매도점을 결정하는 목표 수익률
 8. 매도점 : 평단 * (1 + 목표수익률%) 
 9. 진행률 : 무한매수가 진행된 비율로 (총 매수금액 / SEED) 
 10. 별퍼센트 : 전체 보유수량의 1/4을 매도하는 기준점으로 ( 목표수익률 - (2*진행률)*목표수익률 )
 11. 별퍼센트 매도점 : 평단 * (1 + 별퍼센트%)

## 무한매수법 원리 
 1. 일일매수금액의 절반은 종가가 평단보다 낮은 경우 매수, 일일매수금액의 나머지 절반은 종가가 별퍼센트 매도점보다 낮은 경우에 매수
 2. 매수는 LOC 매수로 주문 - LOC매수: 종가가 장마감 지정가(LOC) 보다 낮은 경우 매수 실행 
 3. 종가가 별퍼센트 매도점을 넘는 경우 보유수량의 1/4 을 매도  : 매도는 LOC 매도로 실행
 4. 전체 매도 : 장중에 매도점에 도달하는 경우 지정가 매도로 보유수량의 3/4을 매도
 5. 전체 SEED를 다 소진했을 때 매도점에 도달하지 못한 경우에는 전체 수량의 1/4을 종가에 강제로 매도
 6. 첫날은 전날 종가의 1.10배 금액을 기준으로 종가에 일일매수금액에 해당하는 수량을 매수

## 추가 Rule 
 1. 종목은 TQQQ 로 진행함.

## 대시보드 최종 출력물
 1. 거래 이력 : 날짜별 매수/매도 내역(구분, 가격, 수량, 금액)과 그에 따른 평단·수량·진행률 변화 기록
 2. MTS 주문 정보 : MTS에 그대로 입력할 매수/매도 주문(LOC/지정가 구분, 가격, 수량)
 3. 차트 : 평단, 매도점, 별퍼센트 가격, 현재가 등을 함께 표시하는 시각화 차트
